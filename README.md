# 📱 Projeto Android - Landing Page

Esta é uma Landing Page responsiva e moderna desenvolvida para explorar as melhores práticas de **HTML5** e **CSS3**. O projeto apresenta a história do mascote do Android de forma limpa, organizada e totalmente adaptável a qualquer tamanho de tela.

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização avançada, uso de variáveis, fontes personalizadas e layouts flexíveis.
- **Design Responsivo**: Adaptado para Desktop, Tablets e Smartphones (Mobile First).

## 💡 O que eu aprendi neste projeto:
- Como trabalhar com **variáveis no CSS** para manter um padrão de cores.
- Implementação de **Media Queries** para garantir que o site fique bonito em qualquer celular.
- Organização de arquivos e diretórios seguindo padrões profissionais.
- Hospedagem e deploy automatizado via **GitHub Pages**.
